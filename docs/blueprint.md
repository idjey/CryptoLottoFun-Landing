# **App Name**: Crypto Lotto Fun

## Core Features:

- Headline Announcement: Display a clear and engaging 'Under Construction' headline.
- Game Description: Provide a brief description of the upcoming cryptocurrency lottery game.
- Email Signup Form: Implement a form where users can sign up with their email to receive launch notifications.
- Falling Crypto Symbols: Animate various cryptocurrency symbols falling from top to bottom, creating a visually appealing rain effect.

## Style Guidelines:

- Dark background using #0B0B0C to set a premium tone.
- Rich gold (#FFD369) for a luxurious and high-value feel.
- Deeper warm gold (#FFAA00) to complement the primary accent.
- Gold gradient (#F5C26B → #FFAA00) for prominent call-to-action.
- Body and headline font: 'Inter' sans-serif font for clear, readable text.
- Use 'Belleza' for headlines and short descriptions, and 'Alegreya' for game instructions.
- Use mono-icons for cryptocurrency symbols, slightly transparent (20–40%).
- Subtle glow effect on crypto symbols using the primary accent color, slower speed at the top of the screen fading towards the bottom.
- Ensure the email signup form is prominently displayed.
